Pierwsza Struktura - Mac OS & Windows

| Visual Studio Code & HTML |

Aby dodać sekcję na stronę internetową, musisz najpierw utworzyć strukturę strony. Poniżej znajdziesz prosty przykład struktury strony w HTML oraz foldery, które mogą być używane dla systemów operacyjnych macOS i Windows.

Pierwsza Struktura HTML

Pierwsza Struktura Plików

W tym przykładzie:

index.html css
Plik główny HTML, który definiuje strukturę strony. Katalog zawierający pliki CSS, takie jak style.css do stylizacji strony.
img js
Katalog zawierający pliki graficzne, takie jak obrazki. Katalog zawierający pliki JavaScript, takie jak skrypt.js do interakcji na stronie.

Wyświetl stronę HTML w swojej przeglądarce

Otwórz zapisany plik HTML w swojej ulubionej przeglądarce. .

MacOS

Dwukrotne kliknięcie pliku lub kliknięcie prawym + lewym przyciskiem touch-pada jednocześnie, a następnie wybierz opcję „Otwórz w apliakcji” poczym wybierz przeglądarkę.

Windows

Dwukrotne kliknięcie pliku lub kliknięcie prawym przyciskiem myszy, a następnie wybierz opcję „Otwórz za pomocą” poczym wybierz przeglądarkę.
