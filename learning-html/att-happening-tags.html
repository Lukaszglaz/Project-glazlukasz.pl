Atrybuty Zdarzeń - HTML

Atrybuty zdarzeń w HTML pozwalają na zdefiniowanie działań wywoływanych przez zdarzenia na stronie internetowej. Oto kilka kluczowych atrybutów zdarzeń z kategorii:

Atrybuty Zdarzeń Okna

Te zdarzenia są wywoływane dla obiektu okna, zazwyczaj stosują się do znacznika <body>.

Atrybut Wartość Opis
onafterprint script script do uruchomienia po wydrukowaniu dokumentu.
onbeforeprint script script do uruchomienia przed wydrukowaniem dokumentu.
onbeforeunload script script do uruchomienia, gdy dokument jest o wyładowywany.
onerror script script do uruchomienia w przypadku wystąpienia błędu.
onhashchange script script do uruchomienia po zmianie części zakotwiczenia w adresie URL.
onload script Wywoływane po zakończeniu ładowania strony.
onmessage script script do uruchomienia po wywołaniu komunikatu.
onoffline script script do uruchomienia, gdy przeglądarka przechodzi w tryb offline.
ononline script script do uruchomienia, gdy przeglądarka przechodzi w tryb online.
onpagehide script script do uruchomienia, gdy użytkownik opuszcza stronę.
onpageshow script script do uruchomienia, gdy użytkownik przechodzi do strony.
onpopstate script script do uruchomienia po zmianie historii okna.
onresize script Wywoływane podczas zmiany rozmiaru okna przeglądarki.
onstorage script script do uruchomienia po aktualizacji obszaru Web Storage.
onunload script Wywoływane po odładowaniu strony lub zamknięciu okna przeglądarki.

Atrybuty Zdarzeń Formularza

Te zdarzenia są wywoływane przez działania wewnątrz formularza HTML.

Atrybut Wartość Opis
onblur script Wywoływane w momencie, gdy element traci fokus.
onchange script Wywoływane w momencie zmiany wartości elementu.
oncontextmenu script script do uruchomienia po wywołaniu menu kontekstowego.
onfocus script Wywoływane w momencie, gdy element zyskuje fokus.
oninput script script do uruchomienia po wprowadzeniu danych przez użytkownika.
oninvalid script script do uruchomienia w przypadku błędu walidacji elementu.
onreset script Wywoływane po kliknięciu przycisku Reset w formularzu.
onsearch script Wywoływane po wpisaniu tekstu w pole wyszukiwania (dla <input="search">).
onselect script Wywoływane po zaznaczeniu tekstu w elemencie.
onsubmit script Wywoływane po złożeniu formularza.

Zdarzenia Klawiatury

Zdarzenia klawiatury są wywoływane w odpowiedzi na interakcje użytkownika z klawiaturą. Oto kilka kluczowych atrybutów zdarzeń klawiatury:

Atrybut Wartość Opis
onkeydown script Wywoływane po wciśnięciu klawisza.
onkeypress script Wywoływane po naciśnięciu klawisza.
onkeyup script Wywoływane po zwolnieniu klawisza.

Zdarzenia Myszy

Zdarzenia myszy są wywoływane w odpowiedzi na interakcje użytkownika z myszą. Oto kilka kluczowych atrybutów zdarzeń myszy:

Atrybut Wartość Opis
onclick script Wywoływane po kliknięciu myszą na elemencie.
ondblclick script Wywoływane po podwójnym kliknięciu myszą na elemencie.
onmousedown script Wywoływane po wciśnięciu przycisku myszy na elemencie.
onmousemove script Wywoływane, gdy wskaźnik myszy porusza się nad elementem.
onmouseout script Wywoływane, gdy wskaźnik myszy opuszcza element.
onmouseover script Wywoływane, gdy wskaźnik myszy najedzie na element.
onmouseup script Wywoływane po zwolnieniu przycisku myszy nad elementem.
onmousewheel script Przestarzałe. Zamiast tego użyj atrybutu onwheel
onwheel script Wywoływane po obróceniu kółka myszy nad elementem.

Zdarzenia Przeciągania

Zdarzenia przeciągania są wywoływane w odpowiedzi na operacje przeciągania elementów. Oto kilka kluczowych atrybutów zdarzeń przeciągania:

Atrybut Wartość Opis
ondrag script script do uruchomienia po przeciągnięciu elementu.
ondragend script script do uruchomienia na końcu operacji przeciągania.
ondragenter script script do uruchomienia, gdy element został przeciągnięty na ważny obszar upuszczania.
ondragleave script script do uruchomienia, gdy element opuszcza ważny obszar upuszczania.
ondragover script script do uruchomienia, gdy element jest przeciągany nad ważnym obszarem upuszczania.
ondragstart script script do uruchomienia na początku operacji przeciągania.
ondrop script script do uruchomienia po upuszczeniu przeciąganego elementu.
onscroll script script do uruchomienia podczas przewijania paska przewijania elementu.

Zdarzenia Schowka

Zdarzenia schowka są wywoływane w odpowiedzi na operacje kopiowania, wycinania i wklejania. Oto kilka kluczowych atrybutów zdarzeń schowka:

Atrybut Wartość Opis
oncopy script Wywoływane po skopiowaniu zawartości elementu przez użytkownika.
oncut script Wywoływane po wycięciu zawartości elementu przez użytkownika.
onpaste script Wywoływane po wklejeniu zawartości przez użytkownika.

Zdarzenia multimediów

Zdarzenia wywołane przez media, takie jak wideo, obrazy i dźwięk (dotyczy wszystkich elementów HTML, ale jest najczęstsze w elementach multimedialnych, takich jak <audio>, <embed>, <img>, <object> i <video>).

Atrybut Wartość Opis
onabort script script do uruchomienia w przypadku przerwania
oncanplay script script do uruchomienia, gdy plik jest gotowy do rozpoczęcia odtwarzania (gdy jest wystarczająco buforowany, aby zacząć)
oncanplaythrough script script do uruchomienia, gdy plik może być odtwarzany aż do końca bez pauzowania w celu buforowania
oncuechange script script do uruchomienia, gdy zmienia się kwestia w elemencie <track>
ondurationchange script script do uruchomienia, gdy zmienia się długość multimediów
onemptied script script do uruchomienia, gdy coś złego dzieje się, a plik nagle staje się niedostępny (na przykład niespodziewane rozłączenie)
onended script script do uruchomienia, gdy multimedia osiągną koniec (użyteczne zdarzenie dla komunikatów typu "dziękuję za słuchanie")
onerror script script do uruchomienia, gdy wystąpi błąd podczas ładowania pliku
onloadeddata script script do uruchomienia, gdy dane multimedialne są załadowane
onloadedmetadata script script do uruchomienia, gdy są załadowane metadane (takie jak wymiary i czas trwania)
onloadstart script script do uruchomienia tuż przed rozpoczęciem ładowania pliku, zanim cokolwiek zostanie faktycznie załadowane
onpause script script do uruchomienia, gdy media są wstrzymane, zarówno przez użytkownika, jak i programowo
onplay script script do uruchomienia, gdy media są gotowe do rozpoczęcia odtwarzania
onplaying script script do uruchomienia, gdy media faktycznie zaczęły się odtwarzać
onprogress script script do uruchomienia, gdy przeglądarka pobiera dane multimedialne
onratechange script script do uruchomienia za każdym razem, gdy zmienia się tempo odtwarzania (na przykład, gdy użytkownik przełącza się w tryb zwolniony lub przyspieszony)
onseeked script script do uruchomienia, gdy atrybut seeked jest ustawiony na false, co oznacza, że szukanie zakończyło się
onseeking script script do uruchomienia, gdy atrybut seeking jest ustawiony na true, co oznacza, że szukanie jest aktywne
onstalled script script do uruchomienia, gdy przeglądarka nie może pobrać danych multimedialnych z jakiegokolwiek powodu
onsuspend script script do uruchomienia, gdy zatrzymano pobieranie danych multimedialnych zanim zostały one całkowicie załadowane z jakiegokolwiek powodu
ontimeupdate script script do uruchomienia, gdy zmieniła się pozycja odtwarzania (na przykład, gdy użytkownik przewija do innego miejsca w materiale multimedialnym)
onvolumechange script script do uruchomienia za każdym razem, gdy zmienia się głośność, co obejmuje ustawienie głośności na "wycisz"
onwaiting script script do uruchomienia, gdy media są wstrzymane, ale oczekuje się, że będą kontynuowane (na przykład, gdy media wstrzymują się, aby buforować więcej danych)

Różne Zdarzenia

Atrybut Wartość Opis
ontoggle script Wywoływane po otwarciu lub zamknięciu elementu <details> przez użytkownika.

Należy pamiętać, że rzeczywiste zastosowanie zależy od konkretnych wymagań projektu w zakresie rozwoju stron internetowych.
