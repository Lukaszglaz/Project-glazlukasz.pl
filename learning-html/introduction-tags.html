HTML

| Hyper Text Markup Language |

Jest to standardowy język znaczników używany do tworzenia stron internetowych. Jest on prosty do nauki i za jego pomocą możemy tworzyć strony internetowe.

Zbudowany jest z różnorodnych elementów które, informują przeglądarkę, o tym jak ma wyświetlić treść oraz precyzyjnie określają fragmenty zawartości, takie jak nagłówki, akapity, hiperłącza i wiele innych.

          
            // Przykładowy początkowy kod html.
            

            

            1
              <!DOCTYPE html>
            

            2
               <html>
            

            3
               <head>
            

            4
                   <title>Moja Strona</title>
            

            5
               </head>
            

            6
               <body>
            

            7
                   <h1>Witaj, Świecie.</h1>
            

            8
               </body>
            

            9
               </html>

Tłumaczenie początkowego kodu HTML

1.

Jest to deklaracja doctype, która informuje przeglądarkę, że strona jest napisana w formacie HTML. Dzięki temu przeglądarka może poprawnie wyświetlić stronę.

2.

Jest to otwierający znacznik HTML. Cała witryna jest zawarta w tym znaczniku.

3.

Ten znacznik zawiera metadane strony, takie jak tytuł wyświetlany na pasku tytułu przeglądarki oraz inne informacje, takie jak połączone arkusze stylów CSS, skrypty JavaScript itp

4.

To jest znacznik tytułu strony. Tekst "Moja Strona" będzie wyświetlany na pasku tytułu przeglądarki.

5.

To zamykający znacznik który, oznacza koniec sekcji z metadanymi.

6.

To jest otwarcie znacznika ciała strony. Wszystkie widoczne treści strony, takie jak tekst, obrazy, linki itp., znajdują się wewnątrz tego znacznika.

7.

To jest nagłówek pierwszego poziomu, oznaczony jako „h1”. Jako najważniejszy tytuł strony pojawia się tekst „Witaj, Świecie.”. Nagłówki są często używane do oznaczania nagłówków i sekcji na stronie.

8.

Jest to zamykający znacznik treści strony, który oznacza koniec widocznej części strony.

9.

To zamykający tag dla elementu który, oznacza koniec dokumentu HTML.

*DODATEK*

Ostrzeżenie:

Cała sekcja <body> jest wyśietlana w przeglądarce. Lecz treść sekcji <title> jest wyśietlana na pasku tytułu przeglądarki lub w zakładce strony.

Czym jest Element HTML

Element HTML jest definiowany przez trzy główne składniki:

- znacznik otwierający

- zawartość

- znacznik zamykający

Czyli:

<Znacznik> Zawartość </Znacznik>

Zakładając, że mamy trzy elementy, czyli element HTML obejmuje całą zawartość od jego znacznika otwierającego do znacznika zamykającego.

Przykłady:

<h2> Mój pierwszy nagłówek </h2>

<p> Mój pierwszy tekst </p>

Ostrzeżenie:

Niektóre elementy HTML, takie jak element </br>, nie posiadają treści i są określane jako elementy puste. Elementy puste nie posiadają znacznika zamykającego.

A jak to wygląda w praktyce?

Celem przeglądarek internetowych jest odczytywanie dokumentów HTML i ich poprawne wyświetlanie.

Przeglądarki nie wyświetlają znaczników HTML, ale wykorzystują je do określenia sposobu wyświetlania dokumentu.
