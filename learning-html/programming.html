Programowanie - HTML

W HTML, tag <script> pozwala na dodawanie skryptów po stronie klienta. Można umieszczać skrypty bezpośrednio w kodzie HTML przy użyciu tagu <script> lub łączyć je z zewnętrznymi plikami poprzez atrybut src w tagu <script>.

Dzięki temu programiści mogą dodawać interakcje, manipulować DOM-em i obsługiwać zdarzenia na stronie internetowej, co umożliwia tworzenie dynamicznych i interaktywnych witryn.

Programowanie HTML

Osadzanie skryptu JavaScript w kodzie HTML:

Osadzanie skryptu JavaScript bezpośrednio w kodzie HTML odbywa się za pomocą tagu <script>.

Ten tag może być umieszczony w sekcji <head> lub na końcu sekcji <body>. Osadzony skrypt jest wykonywany w momencie wczytywania strony przez przeglądarkę.

Korzystanie z zewnętrznego pliku JavaScript

Czasami jest praktyczne przechowywanie kodu JavaScript w osobnych plikach, zwłaszcza gdy skrypty są rozległe lub mają być używane na wielu stronach. Aby tego dokonać, możesz użyć atrybutu src w tagu <script>, aby wskazać ścieżkę do zewnętrznego pliku zawierającego kod JavaScript.
