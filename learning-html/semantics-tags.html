Semantyka - HTML

Semantyka HTML to sposób nadawania znaczenia elementom strony internetowej.

Na przykład, element <header> definiuje nagłówek dokumentu, <nav> określa łącza nawigacyjne, a <article> definiuje artykuł.

Dzięki semantyce, struktura dokumentu staje się bardziej zrozumiała zarówno dla przeglądarek, jak i dla programistów.

Semantyka HTML

Nagłówek

<header>

Ten element definiuje nagłówek dokumentu lub sekcji. Zazwyczaj zawiera on tytuł, logo, a także inne elementy, które mają być wyświetlane na górze strony lub w sekcji nagłówkowej.

Główny

<main>

Element ten określa główną treść dokumentu. W jednym dokumencie HTML może być tylko jeden element <main>, który obejmuje główne treści strony, wyłączając nagłówek, stopkę, menu nawigacyjne i inne elementy.
